## 14. お問い合わせ

### セクションタイトル
```
お問い合わせ / まずはお気軽にご相談ください / 無料相談はこちら
```

### リード文
```
[サービス名]に関するご質問・ご相談は、お気軽にお問い合わせください。
[営業時間内は電話でもご対応しております。]
通常 [1〜2営業日] 以内にご返信いたします。
```

### 問い合わせ方法一覧

| 方法 | 詳細 | 受付時間 |
|------|------|---------|
| Webフォーム | 下記フォームよりご送信ください | 24時間受付 |
| 電話 | [00-0000-0000] | [平日 9:00〜18:00] |
| メール | [info@example.com] | 24時間（返信：翌営業日） |
| LINE | [友だち追加ボタン] | 24時間（返信：翌営業日） |

### お問い合わせフォーム 項目定義

| フィールド | 種別 | 必須 | バリデーション |
|-----------|------|------|-------------|
| お名前 | テキスト | ✅ | 最大50文字 |
| フリガナ | テキスト | 任意 | カタカナのみ |
| 会社名・屋号 | テキスト | 任意 | 最大100文字 |
| メールアドレス | email | ✅ | メール形式 |
| 電話番号 | tel | 任意 | 数字・ハイフン |
| お問い合わせ種別 | セレクト | ✅ | [サービス全般 / 料金 / 見積もり / その他] |
| お問い合わせ内容 | テキストエリア | ✅ | 最大1000文字 |
| 個人情報への同意 | チェックボックス | ✅ | チェック必須 |
| 送信ボタン | button | - | - |

### 送信後の処理
- **サンクスページ / メッセージ**: 「お問い合わせありがとうございます。[1〜2営業日]以内にご連絡いたします。」
- **自動返信メール**: 受付確認メールを送信（件名：「[事業者名]お問い合わせを受け付けました」）
- **管理者通知メール**: 担当者に転送
- **Google Analytics**: コンバージョンイベント設定推奨

## 付録B: SEOチェックリスト

- [ ] タイトルタグ（30〜35文字）に主要キーワードを含める
- [ ] メタディスクリプション（120〜140文字）を設定
- [ ] h1は1ページ1つ、h2〜h6は階層的に使用
- [ ] 画像にalt属性を設定
- [ ] 内部リンク（サービス→事例→お問い合わせ）を設置
- [ ] パンくずリスト（下層ページ）を設置
- [ ] 構造化データ（LocalBusiness / FAQ / Review）を設定
- [ ] サイトマップ（sitemap.xml）を作成・送信
- [ ] robots.txtを設定
- [ ] Core Web Vitals（LCP・FID・CLS）を最適化
- [ ] モバイルフレンドリー対応（レスポンシブデザイン）
- [ ] HTTPS対応（SSL証明書）
- [ ] Googleビジネスプロフィールと情報統一
