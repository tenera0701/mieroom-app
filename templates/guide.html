ミエルーム 使い方ガイド

各機能を 「どこに何を入力して、どうすると、どうなるか」 という手順で、実際の画面とあわせて説明します。

左の目次から見たい機能へジャンプできます。上の検索ボックスにキーワード（例：自動返信入金QR）を入れると、関係する項目だけに絞り込めます。画面の画像はクリックすると拡大します。

はじめに
基本

0. ログインと画面の見方

まずは全体の画面構成と、どの画面でも共通する操作を覚えましょう。

ログインの手順

  1. ログイン画面で ユーザー名（またはメールアドレス） と パスワード を入力します。
  2. ログイン ボタンを押すと管理画面（売上分析）が開きます。

画面の各パーツ

  1. 左サイドバー：各機能への入口です。クリックで画面が切り替わります。事務作業各種設定 など▼が付いた項目は、クリックすると下にメニューが開きます。
  2. 左上の店舗切替（「店舗管理」の下のプルダウン）：複数店舗がある場合、表示する店舗をここで切り替えます。多くの管理表・分析は「いま選んでいる店舗」のデータを表示します。
  3. 各画面の上部にある月切替（‹前月 2026年6月 次月› 今月）：対象月を変更します。売上・反響・接客などは月単位で集計されます。
  4. 右上のベル🔔：未承認の入金報告・未返信メール・新着お問い合わせなどのお知らせが一覧で出ます。各項目をクリックすると該当画面へ移動します。
  5. 右上の「ヘルプ」：このガイドを開きます。「改善提案」からは、使いづらい点や要望を本部へ直接送れます。
困ったときは、まず 店舗 と 対象月 が正しいかを確認してください。「データが出ない」原因の多くは、別の店舗・別の月を見ていることです。
分析・ダッシュボード
分析

1. 売上分析

サイドバー：売上分析

当月の売上・利益・契約数・経費を、ひと目で確認できるトップ画面です。

  1. 上部4枚のカードで 今月の売上 / 利益 / 契約数 / 経費合計 を確認します。各カードには前月・前年同月との比較と、月別の推移グラフが付いています。
  2. 総売上は「顧客管理表」の入金額（入金月ベース）を自動集計したものです。数字を増やすには、顧客管理表で入金報告→承認を行います（→5. 顧客管理表）。
  3. 下部の スタッフ別今月実績 で、担当者ごとの反響・来店・契約・売上を確認できます。編集 から数字の手修正も可能です。
  4. 右下の 反響ファネル で「反響→来店→内見→申込→契約」の絞り込み状況を確認します。
  5. 右上の ＋ 数字を入力する から、KPI実績を直接入力することもできます。
分析

2. 本部ダッシュボード

サイドバー：本部ダッシュボード

複数店舗を 横断 して、全社の数字・店舗比較・アラートを見る経営者向けの画面です。

  1. 上部カードで 全店舗合計 の売上・利益・経費・反響数・申込数・契約数を確認します。右上の月で対象月を変えられます。
  2. 各店舗 今月の実績 の表で、店舗ごとの数字を並べて比較できます。
  3. 店舗ランキング：売上利益反響数 などのタブを切り替えると、その指標での順位が見られます。
  4. 店舗別月次比較のグラフで、期間を指定して各店舗の推移を重ねて確認できます。
  5. 一番下の アラート で、目標未達の店舗や売上急落の店舗が自動で赤く表示されます。
この画面は全店舗のデータをまとめて見るため、左上の店舗切替の影響を受けません。
分析

3. 営業分析

サイドバー：営業分析

店舗・スタッフ別の成約率や売上KPIを分析します。スタッフ評価や目標管理に使います。

  1. 上部カードで 総売上・平均成約率・申込数・契約率 と、ライフライン/火災保険/引越しの成約率 を確認します。
  2. スタッフ別KPI の表で、各スタッフの反響→接客→申込→契約の流れと、売上・目標・達成率 を一覧できます。達成率はバーで色分け表示されます。
  3. 編集 ボタンから、各スタッフの目標額や実績を直接修正できます。
  4. 下部の 月次売上推移 グラフで、売上（棒）と申込件数（折れ線）の推移を確認します。
  5. 右上の Excelインポート から、外部データの取り込みも可能です。
分析

4. 反響分析

サイドバー：反響分析

どのポータル（媒体）からどれだけ反響・申込が出ているかを分析し、広告費の効果を見える化します。

  1. 上部カードで 総反響・来店・申込・売上・AD（広告費） の当月合計を確認します。
  2. 媒体別カード（HP・紹介・SUUMO・HOME'S 等）で、媒体ごとの反響件数と前月比、CVR（成約率）を見ます。
  3. 媒体別詳細の表で、反響・来店・申込・売上・広告費・CPA（1件あたり獲得単価）などを横並びで比較できます。
  4. 下部の 月次トレンド グラフで、反響数・申込数・売上の推移を確認します。
  5. 数字は ＋ 新規入力 や、反響管理表・自動取込の実績から積み上がります。
CPAが高い媒体は「費用のわりに決まっていない」サインです。媒体ごとの数字を見て広告予算の配分を見直しましょう。

管理表

5. 顧客管理表（入金・売上）

サイドバー：営業分析 › 顧客管理表

申込ごとの金額（仲介手数料・AD・その他費用）と入金状況を管理します。ここで承認された入金額が 売上分析の総売上 になります。

新しい申込を登録する

  1. 右上の ＋ 新規入力 を押すと入力フォームが開きます。
  2. 下の各欄を入力します。申込日（必須）、担当スタッフ、媒体、物件名、号室、お客様名、各種金額（仲介手数料・AD金額・その他費用）など。
  3. ステータス（申込／契約／キャンセル／キャンセル振替）を選びます。
  4. 付帯サービス（ライフライン・引越し・火災保険）に該当すればチェックします。営業分析の成約率に反映されます。
  5. 保存 を押すと一覧に追加されます。

入金を報告して売上に計上する

  1. 一覧の入金欄にある 報告 ボタンを押し、入金（予定）日を選んで報告します。
  2. 店長・オーナーが 承認 すると、その日付で入金が確定し、入金月ベースで売上に集計されます。
  3. 未承認の入金報告は、右上のベル🔔のお知らせにも表示されます。
画面下部には 入金待ち一覧 と 売上一覧 があり、「まだ入っていないお金」と「入ったお金」を分けて管理できます。
管理表

6. 反響管理表

サイドバー：営業分析 › 反響管理表

ポータルから来た反響（お問い合わせ）を一覧で追客管理します。表のセルをクリックして直接編集できます。

基本の使い方

  1. 右上の ＋ 新規追加 で行を追加するか、自動取込（→19）で反響が自動で入ります。
  2. 担当者・反響日・状況・顧客名・媒体・手段・番号有無・初回対応日・追客日程・メモなどの セルを直接クリックして編集します。
  3. 上部の検索ボックスで、顧客名・媒体・担当で絞り込めます。上のサマリーカードで総反響数や電話番号あり件数を確認できます。

状況タグと行の色

  1. 状況列のセルでタグ（追客中・申込・内見予定・終了など）を選びます。選んだタグの色で行の背景色が変わります。
  2. 状況見出しの ⚙ から、タグの追加・削除と、各タグの行の色を設定できます。
  3. お客様から返信が来て未返信の行は、左端に 赤いライン が付いて目立ちます。

反響からメールでやり取りする

  1. 各行の ✉️メール ボタンでチャット画面が開き、お客様とメールでやり取りできます。
  2. テンプレ挿入（📋）・画像/PDF添付（📎）が使えます。送信は Shift+Enter（Enterは改行）または右下の➤ボタン。
  3. お客様がメールを開くと「既読」が表示され、返信が来ると未返信としてアラート表示されます。
管理表

7. 接客管理表

サイドバー：営業分析 › 接客管理表

来店・電話・メールなどの接客対応を1件ずつ記録し、申込率を見ます。

  1. 右上の ＋ 新規追加 で行を追加します。
  2. 日付・担当者・反響媒体・お客様名・対応種別（来店/電話/メール/再来）・来店数・状況・メモを、セルをクリックして入力します。
  3. 上部サマリーで 総接客数・申込数・申込率・他決・キャンセル を確認できます。
  4. 上部の検索や、右上の月・スタッフ絞り込みで対象を切り替えられます。
管理表

8. カレンダー

サイドバー：営業分析 › カレンダー

来店予約や予定をカレンダーで管理します。Web予約（→23）で入った予約も自動でここに表示されます。

  1. 左上の ＋ 作成 から予定を新規登録します。日時・お客様名・担当スタッフ・メモを入力します。
  2. 右上の 月週日 で表示を切り替えます。
  3. 左の スタッフ（色分け） でチェックを入れると、担当者ごとに色分けして表示されます。チェックを外すとそのスタッフの予定を隠せます。
  4. カレンダー上の予定をクリックすると、内容の確認・編集ができます。

9. 物件コンバータ

サイドバー：物件コンバータ

物件を 一度入力するだけ で、SUUMO／HOME'S／at home 等の規定フォーマットへ一斉出力できる機能です。AIによる物件コメント自動生成も搭載。

物件を登録する

  1. 右上の ＋ 新規物件 を押すとフォームが開きます。
  2. 上部のタブ（基本／所在地・交通／賃料・契約／建物・規模／設備・条件／取引・備考／画像）を順に切り替えて入力します。
  3. 「基本」タブで取引区分・物件種別・物件名・部屋番号・ステータス（下書き/公開など）を設定します。
  4. 入力後 保存 すると一覧に並びます。

ポータル用に一斉出力する

  1. 一覧で出力したい物件のチェックを入れます。
  2. 右上の 📤 選択した物件を一斉出力 を押し、出力先ポータルの形式でデータを書き出します。
物件コンバータ・間取り作成・画像編集は オプション機能 です。サイドバーに表示されない場合は、ご契約プランをご確認ください。

10. 間取り作成

サイドバー：事務作業 › 間取り作成

物件の間取り図をツール上で作成・保存・印刷できます。フォルダで物件・部屋タイプ別に整理できます。

  1. ＋ 新規作成 で作図画面を開きます。壁・ドア・窓・設備（コンロ・バス・トイレ等）を配置して間取りを描きます。
  2. 整理用に ＋ フォルダ作成 でフォルダを作り、間取りを分類できます。
  3. 手元の 間取り画像をなぞって描きたいときは「下絵」機能を使います。画像を薄く透かして表示し、その上をなぞって作図できます。
  4. 作成した間取りは保存され、一覧から開き直し・印刷ができます。

11. 画像編集

サイドバー：事務作業 › 画像編集

物件画像に自社ロゴや帯を付けるなど、簡単な画像加工ができます（間取り作成と同じオプションに含まれます）。

  1. 編集したい 物件画像をアップロード します。
  2. ロゴ・帯・文字などを重ねて配置し、見栄えを整えます。
  3. 加工後の画像を ダウンロード して、ポータル掲載や資料に使います。

12. 契約管理

サイドバー：事務作業 › 契約管理

契約手続き中のお客様を管理し、契約書類の作成につなげます。

  1. 申込済みのお客様が一覧に並びます。各行で契約日・物件・お客様情報などを管理します。
  2. 各行の操作ボタンから、お客様情報をもとに 契約書類を作成 できます（書式は 21. 契約フォーマット で登録したものを使用）。
  3. 上部の検索で、お客様名・物件名から目的の行を素早く探せます。

13. 顧客管理（入居後）

サイドバー：事務作業 › 顧客管理

契約が終了し入居が完了したお客様の管理ファイルです。アフターフォローや過去履歴の参照に使います。

  1. 入居完了顧客一覧 に、契約・入居が済んだお客様が蓄積されます。
  2. 上部の検索ボックスに、お客様名・物件名・号室を入れて該当のお客様を探します。
  3. 右上の 年月へ移動 で、過去の月のデータをさかのぼって確認できます。

14. 日報

サイドバー：事務作業 › 日報

スタッフが毎日の営業活動を記録します。タスクの抜け漏れ防止と、日々の振り返りに使います。

  1. 右上で 日付 と 記入するスタッフ を選びます。
  2. タスクチェック：来店前日連絡・当日連絡・申込管理機能入力など、その日やることにチェックを入れます。
  3. 接客記録：＋ お客様追加 でその日対応したお客様を記録し、今日の申込数 を入力します。
  4. 明日の接客予定 と その他メモ を記入します。
  5. 最後に 日報を保存する を押します。上部の 日報一覧 タブから過去の日報を確認できます。

15. 有給管理

サイドバー：事務作業 › 有給管理

スタッフの有給・休暇を申請・記録し、残日数を管理します。

  1. 右上の 休取入力 から休暇を登録します。スタッフ・日付・種別（有給/半休/欠勤）・日数・メモを入力します。
  2. 上部 年次有給残日数 で各スタッフの残りを確認できます。
  3. 休暇記録一覧 に登録済みの休暇が並びます。ステータス（承認済など）を管理し、各行の操作ボタンで編集・削除ができます。
  4. 右上のプルダウンで、スタッフ別・種別で絞り込めます。

16. 経理・PL管理

サイドバー：事務作業 › 経理・PL管理

月次内訳・費用内訳・利益率を管理する損益（PL）画面です。

  1. 上部カードで 売上・粗利・営業利益・経費合計・営業利益（粗利） を確認します。月切替で対象月を変えられます。
  2. PL明細の表で、収益（売上）と費用（広告宣伝費の媒体別内訳・人件費・その他）を当月／前月／前年同月で比較できます。
  3. 各項目の 編集 から金額を修正できます。広告費の内訳（SUUMO・HOME'S 等）も入力できます。
  4. 下部の 月次利益推移 グラフで、売上・経費・利益の推移を確認します。
  5. 右上の Excelレポート で帳票を書き出せます。

17. 社内チャット

サイドバー：チャット

社内メンバーとメッセージでやり取りします。全社・店舗別・グループのチャンネルがあります。

  1. 左の チャンネル一覧 から話す相手を選びます。全社（全員）・店舗別（新宿店・横浜店・渋谷店など）があります。
  2. 下の入力欄にメッセージを書き、Shift+Enter または ➤ で送信します。文中に @ を入れるとメンション（名指し通知）できます。
  3. オーナー・店長は ＋ グループ から、特定メンバーだけのグループチャンネルを作成できます。
  4. 右上の メモアルバム で、共有メモや画像をまとめて見られます。通知をONにする で新着通知を受け取れます。
画像・PDFの添付と 2年間の保存 は チャットProプラン（オプション）でご利用いただけます。通常はテキストのみ・60日保存です。

メール設定

18. メールテンプレート

サイドバー：各種設定 › メールテンプレート

お客様へ送るメールのひな形（文面）を作っておく場所です。反響メールの返信や自動返信で使います。

テンプレートを作る

  1. ＋ 新しいテンプレートを作成 を押すとエディタが開きます。
  2. 件名と本文を入力します。HTML（太字・色・リンク・リスト）で装飾できます。
  3. 本文に 差し込み文字 を入れると、送信時に実データへ自動で置き換わります。例：#name#（お客様氏名）#お問い合わせ物件名##会社名##会社電話番号##公式LINE#
  4. 保存すると一覧に追加され、編集削除 で後から修正できます。
差し込み文字の中身（会社名・電話など）は 26. 会社情報 に登録した内容が使われます。先に会社情報を埋めておきましょう。
メール設定

19. メール自動取込

サイドバー：各種設定 › メール自動取込

反響メールが届くGmailを連携し、届いた反響を自動で反響管理表に取り込みます。

接続する（2つの方法）

  1. かんたん：Googleで連携 … 上部の Googleで連携する を押し、画面の案内に従ってGoogleアカウントを許可します。
  2. アプリパスワード方式 … メールアドレスと、Googleで発行した16桁の アプリパスワード を入力し、接続テスト で確認します。作り方は画面下部の 🔑アプリパスワードの作り方 を開くと手順が出ます（2段階認証ON→アプリパスワード発行→貼り付け）。

自動取込をONにする

  1. 取込設定 の 自動取り込みを有効にする をONにします。届いた反響がリアルタイムで反響管理表に入ります。
  2. ポータル登録 で「差出人アドレス → 媒体名」を登録すると、確実に媒体ごとに振り分けられます（例：@suumo.jp → SUUMO）。＋ 行を追加 で複数登録できます。
  3. 最後に 設定を保存。動作を試したいときは 今すぐ取込 を押します。
接続に失敗するときは、メールアドレスの打ち間違いか、通常のログインパスワードを入れている（必ず16桁のアプリパスワードを使う）ことが多いです。
